Summary

3D-LiDAR Annotation Tool is an app to annotate and extract point cloud data from a binary extension file. The app has a built-in .pcd2.bin conversion tool to convert the .pcd files to .bin files before annotating. The annotated data can be exported and saved as a .json file, which has all the labeling boxes information including 3D-centerpoints, width, length, height, and orientation.
